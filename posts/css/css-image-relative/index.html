[CSS] 여러 개의 이미지 & 글씨 겹쳐 보이게 하기

안녕하세요!

이번 포스팅 에서는 CSS를 사용하여 여러 개의 이미지나 글씨를 화면에 겹쳐 보이게 하는 방법을 소개해 드리겠습니다.🌈

1. 개요

웹 페이지에서 흔히 볼 수 있는 배너 같은거를 만들 때 사용 되는 방법 입니다. 배너 위에 타이틀이 있고, 서브 타이틀이 있는 배너 다들 한번씩은 보셨죠?
배너 이미지, 타이틀 이미지, 서브 타이틀 이미지 3개가 주어 졌을 때 여러 개의 이미지를 겹쳐서 보이게 하는 방법을 알려 드리겠습니다.

2. CSS 문법 요소

display: relative;
display: absolue;
z-index: 10;

오늘 배우고 적용할 CSS 요소 입니다. 이것만 확실히 알면 이미지 & 글자 겹쳐 보이게 하기 완전 쉽게 적용 할 수 있어요! (야나두?..🥲)
CSS로 배너 만들기 왕이 될 수 있어요!!(?)

3. 적용 방법

CSS로 만들어 보기 전에 배너로 만들어 볼 예시 사진을 보여 드릴께요.
🗣 이미지 출처: https://blog.naver.com/s0813se (맞아요 저희집 강아지예요..👀)

대충 파워 포인트로 만들어 보았는데 이런 형태의 배너를 css를 이용해서 만들어 볼께요!

4. 마치며

  • 웹 개발 시에 쿠키 값을 유지하고, 공유 하는 작업은 처음 해봐서 블로그에 적게 되었습니다. 생각보다 간단 하죠?!🤭
  • 생각 보다 간단 하지만 구글링하고 삽질하는데 시간이 꽤 오래 걸려서…저 같은 분들에게 도움이 되셨으면 좋겠어요!👍🏻
