[React] React 프로젝트 생성 & 실행 방법

안녕하세요!

요즘 React.js를 이용해서 프로젝트를 진행 중이라 다시 시작하는 마음으로 앞으로 튜토리얼 형식으로 포스팅을 정리할 예정입니다.
그래서 이번 포스팅 에서는 React 프로젝트를 생성하고, 실행하는 방법에 대해 알아보도록 하겠습니다.!🤔

  • 저는 Mac OS 환경에서 Intellij IDE를 사용하고 있습니다. 본인이 사용하는 툴에 따라 아래의 방법을 따라 하시면 됩니다!

프로젝트 생성

  1. 터미널에서 React 앱을 생성하기 위한 모듈을 설치 합니다. -g 옵션으로 시스템 공통 폴더에 설치 됩니다.

    npm install -g create-react-app

  2. 위의 명령어 실행 시, 아래처럼 권한 문제로 설치에 실패 한다면 sudo 명령어로 권한을 부여해 줍니다.

  3. 명령어 실행 

    sudo npm install -g create-react-app

  1. 명령어 실행이 완료 되면, 프로젝트 이름을 입력해서 프로젝트를 생성 합니다.

  2. React 프로젝트는 npm 모듈 설치 하듯이 명령어 한 줄이면 알아서 프로젝트 구조가 생성이 되어 편리합니다!

    create-react-app "프로젝트 이름"

  3. 자신이 생성한 프로젝트 이름 해당 경로로 이동하여 프로젝트가 제대로 생성이 되었는지 확인 합니다.

    cd "프로젝트 이름"

프로젝트 실행

  • 생성한 프로젝트의 루트 경로에 package.json 이라는 파일이 생성 되었습니다.
  • script에 작성된 명령어로 프로젝트를 실행 할 수 있습니다.

  • 프로젝트 실행 방법은,
    (1) 왼쪽의 재생 버튼을 눌러 프로젝트를 실행 합니다.
    (2) 터미널에서 명령어를 입력 합니다.

    npm run start
  • 명령어를 실행하면 터미널에 프로젝트가 생성 되었다고 알려 줍니다.
  • localhost:3000 (기본포트는 3000) 주소로 들어가면 프로젝트가 실행 된것을 확인 할 수 있습니다!
